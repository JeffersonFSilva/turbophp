Class TTpFileList

Description
Description | Vars (details) | Methods (details)

Collect information about items in a file system directory.

Builds an array of records about entries in a given file-system directory.

Set the $Folder field to the pathname to examine.

The list is automatically built when Generate is called, or the list can be built at any time by calling the ListFiles method.

The $Files field is an array of file names (or complete paths if $FullPath is true) in the directory.

The $Infos field is an array of information records. Both the $Files and $Infos arrays contain $Count records.

The list can be sorted by calling the Sort method and specifying one of the info record field names (see $Infos documentation).

mixed $Infos (line 1617)

Array of information records.

Each record is an array with the following fields:

  • name => name of the file
  • size => size of the file in bytes
  • time => modification date for the file
  • dir => true if the file is a directory
  • ext => file's extension
E.g., the extension of the third entry in the directory is accessible as: 
  1. $this->Infos[2]['ext']

string $OnFilter (line 1637)

List filter event.

Called once for each item found while enumerating the folder. The event handler can examine the $File property for the name of the current file. If the current file should be filtered out of the list, the handler sets $File to the empty string ('').
