Class TTpTable

Description
Description | Vars (details) | Methods

A dynamic table.

A table whose layout and content are specified at runtime (i.e. using PHP).

Set the number of cells with the $Rows and $Cols properties.

Each time a cell is generated, the $OnCell event is fired. The row number is held in $Row and the col number in $Col (zero-based). Use the $OnCell event to specify the $RowSpan and $ColSpan for the cell, to setup $CellAttributes, and assign $Content.
