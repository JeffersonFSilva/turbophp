Class TTpRequestVariable

Description
Description | Vars (details) | Methods (details)

Stores data parsed from a request variable.

Request variable (GET, POST, and COOKIE) data are parsed into TTpRequestVariable objects. Information encoded into the variable text, such as hit-coordinates and control indices are parsed into fields. See TTpPage.Inputs.
