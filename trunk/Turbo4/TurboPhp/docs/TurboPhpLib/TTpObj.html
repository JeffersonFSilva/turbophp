Class TTpObj

Description

Base class for TurboPhp objects.

A TTpObj is created for each marked-up HTML element in the template.

			array
			$Attributes
						(line 916)
		
	


	

Array of HTML attributes (name/value pairs).


Attributes are encoded using XHTML standards (i.e. attribute names  are all lowercase).
Change attributes by setting array values, e.g.,  
  1. Attributes['bgcolor'] = 'green';
    2. 

  Clear an attribute by  
  1. Attributes['bgcolor'] = '';
    2. 

  or  
  1. unset($object->Attributes['bgcolor']);
    2. 

  Attributes that don't have values should be encoded XHTML style, e.g.  
  1. Attributes['readonly'] = 'readonly'
    2. 

  Note: avoid setting the style attribute using Attributes  array. Instead, put style entities into the Styles array.

Name of OnGenerate event function.


This event is fired from Generate before output  is generated for this object.  This event may be fired multiple times if the object is used more than  once in the page. For example, a label object in a TTpRepeater  will be generated once for each repeat.
