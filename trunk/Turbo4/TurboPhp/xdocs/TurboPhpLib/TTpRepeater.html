Class TTpRepeater

Description
Description | Vars (details) | Methods

A container which can output itself multiple times.

A repeater will generate itself as a series of rows until $Count iterations have been completed or $Stop becomes true. The $OnRepeat event is fired once before each iteration. Controls inside the repeater will fire OnGenerate events every iteration. The $Index property indicates the current iteration (zero based, i.e., the first iteration has an $Index of zero).

mixed $OnRepeat (line 2069)

Event that is fired before each repeat.

The $OnRepeat event is fired once before each iteration.

The $Index property indicates the current iteration (zero based, that is, the first iteration has $Index = zero).

Set $Stop to true inside this event to stop generation immediately.

$Stop will already be true if $Index = $Count and $Count > 0. In this case, you may set $Stop to false to continue repeating

Note that controls inside the repeater will fire OnGenerate events each iteration.

Boolean $Stop (line 2052)

Set true to stop the repeater.

The repeater will iterate until there have been $Count interations or until $Stop becomes true. If $Count is zero, the repeater will repeat indefinitely until $Stop is true.
