array $Options (line 1833)

Array of options to display.

The $Selected property always refers to a key of this array. The key could be an integer index, or a string if if $Options is a hashed array.

mixed $Selected (line 1842)

Key of the selected item in the Options array.

Read this property to discover the selected value. Write this property to change the selection. To read/write the selected {@var value} use the GetSelectedValue or SetSelectedValue methods.

string $Source (line 1848)

Reference to a TTpListSource object which will supply the option array.
