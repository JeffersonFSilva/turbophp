Class TTpApp

Description
Description | Vars (details) | Methods (details)

Application framework object.

Implements the TurboPhp application framework, including parsing the input template, handling input, and generating HTML. The behaviour of the application is customized by attaching event handlers.

		Interpolate (line 544)
	
 
	
	

Expand all object and block macros in the input text.


Macro patterns of the form {%Macro} in the input text are  matched against object names, then block names.
Macros matching objects are replaced by text provided by the object's  TTpObj.Generate() function.
Block macros are replaced by the stored block value.
Note: Expanded macro patterns are interpolated recursively.

	
    
				
  • return: Input text with objects and macros recursively expanded.
    •
